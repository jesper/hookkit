hookkit.pyÂ¶

hookkit.py is the first point of contact for when git itself interacts with HookKit. hookkit.py queries the Configuration File for which scripts it should execute, how, and when.

This program should not be executed by the user, but rather purely from git - via a link created by install_hooks.py.
